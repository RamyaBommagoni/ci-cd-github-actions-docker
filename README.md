# CI/CD Pipeline Automation using GitHub Actions

## Overview
This project demonstrates a CI/CD pipeline that automates build, test, Docker image creation, and deployment using GitHub Actions.

## Tools Used
- GitHub Actions
- Git
- Docker
- Linux

## Pipeline Workflow
1. Code commit triggers the pipeline
2. Build and test stages are executed
3. Docker image is built
4. Application is deployed
